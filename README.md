## Getting Started

- `cd ~ && git clone https://github.com/NarcoNik/setup.git`

### Installing debs programm

- PowerShell, Discord, Github Desktop, Anydesk, Lutris

- `sudo chmod +x ./setup/Linux/debs.sh`
- `./setup/Linux/debs.sh`

### Installing Nvidia drivers

- `sudo chmod +x ./setup/Linux/nvidia.sh`
